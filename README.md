# code-review

Base repository for candidates to review and refactor. A Node.js project built with TypeScript and ES Modules.

## Installation

```bash
npm install
```

## Running the Project

```bash
# Run in development mode
npm run dev

# Build for production
npm run build

# Run the compiled code
npm start
```

## Testing

```bash
# Run tests
npm test

# Run tests in watch mode
npm run test:watch

# Run tests with coverage
npm run test:coverage
```

## Code Quality

```bash
# Check for linting errors
npm run lint

# Format code
npm run format
```
